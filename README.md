# Daily Reflection Tree

A deterministic, rule‑based end‑of‑day reflection tool for employees, grounded in three psychological axes: **Locus of Control**, **Contribution vs Entitlement**, and **Radius of Concern**.

## Features

- Fully deterministic – no AI at runtime (no hallucination)
- 50+ nodes: start, question, decision, reflection, bridge, summary, end
- CLI agent written in Python (standard library only)

## How to run

```bash
python agent.py
```
